# Exercícios de Programação em Python

## 1. Gerador de Senhas Aleatórias

Crie um programa que gere **senhas aleatórias** com letras, números e símbolos.  
O usuário deve escolher o **tamanho da senha** para criar senhas seguras automaticamente.

---

## 2. Consulta de Usuário Fictício via API

Crie um programa que acesse uma **API** para buscar um usuário fictício aleatório.  
O programa deve exibir:
- Nome  
- E-mail  
- País  

Caso houver erro na conexão, mostrar uma **mensagem de falha**.

---

## 3. Consulta de CEP via API

Crie um programa que consulte informações de um **CEP** na API.  
O programa deve retornar:
- Logradouro  
- Bairro  
- Cidade  
- Estado  

Caso o CEP não existir ou houver erro na requisição, mostrar uma **mensagem de falha**.

---

## 4. Consulta de Moeda em Relação ao Real (BRL)

Crie um programa que realize consultas de moedas em relação ao **Real (BRL)** usando uma API.  
O programa deve mostrar:
- Valor atual  
- Valor máximo  
- Valor mínimo  
- Data/hora da última atualização  

Caso a moeda não existir ou houver erro na requisição, retornar uma **mensagem de erro**.
