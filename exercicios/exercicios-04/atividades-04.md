# Exercícios de Programação em Python

---

## 1. Calculadora Básica

Criar um código que faça uma calculadora que tenha as operações básicas:

- Adição (`+`)
- Subtração (`-`)
- Multiplicação (`*`)
- Divisão (`/`)

---

## 2. Registro de Notas

Criar um código que registre as notas de alunos e calcule a média da turma.

---

## 3. Verificação de Senha Segura

Criar um código que verifique se uma senha digitada pelo usuário atende a critérios básicos de segurança:

- Deve ter **pelo menos 8 caracteres**
- Deve conter **pelo menos um número**

---

## 4. Classificação de Números

Criar um código que analise números digitados pelo usuário, classificando-os como:

- **Pares** ou **ímpares**
- E contabilize **quantos de cada tipo** foram inseridos
